# Material design icons

Google が公開している、マテリアルデザイン準拠の公式アイコンセットです。
https://fonts.google.com/icons で閲覧できます。

## デモ
https://code4fukui.github.io/material-design-icons/

## 機能
- アイコンを検索して表示できます
- カテゴリ別にアイコンを表示できます
